# OpenClaw VS Code Extension

OpenClaw AI Assistant for VS Code - Multi-agent conversation with Gateway integration.

## Usage

1. Open the OpenClaw view in the sidebar (Activity Bar -> OpenClaw icon)
2. The extension auto-connects to the Gateway on startup using `~/.openclaw/openclaw.json`
3. Select an agent from the dropdown
4. Start chatting

## Commands

- **Refresh** - Refresh the chat view
- **Clear Chat** - Clear the chat history
- **Connect to Gateway** - Manually connect to the Gateway server
- **Disconnect from Gateway** - Disconnect from the Gateway server
